COMANDOS HTML

Significados das "tag" do HTML

A palavra 'tag' significa o processo de rotular, classificar e etiquetar as funções/comandos.

Há diversas classes que separam a semântica das tag's

METADADOS:

Os metadados possuem a função de guardar várias funções sobre a página no que tange ao: estilo, scripts, ferramentas de pesquisa, etc.


Tag's Metadados:
  1. '<'base'>' => Endereço URL utilizada por todos os endereços relativos contidos dentro de um documento.

  2. '<'head'>' => HEAD = Cabeça. Providencia informações gerais sobre o documento.
    3. EX: título, links para script e folhas de estilo.
  3. '<'link'>' => Ocorre uma relação entre o documento atual e um recurso externo, isto é, vincular folhas de estilos/páginas.
  4. '<'meta'>' => Define qualquer informação de metadados que não podem ser definidos por outros.
    5. EX: meta charset; meta name="".
  5. '<'style'>' => Introduz informações de estilos
  6. '<'title'>' => Título do Documento/Projeto

SEPARAÇÃO DE CONTEÚDO:

Permitem organizar o conteúdo em partes lógicas. Usado para criar esboços do conteúdo da página.

EX: Navegação por cabeçalho e rodapé.
Cabeçalho para identificar seções do conteúdo.
